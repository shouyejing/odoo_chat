Setting up...

First things first: SignUp with Slack

Now go to https://api.slack.com/web and scroll down to the Authentication section.

If your team doesn't have a token you should generate one: write it down

That's pretty much it

After installing the addon a form will let you set the data

Featuring...

Data definitions

Various Schemas that will allow you to map your Odoo models as Slack's.

  • User
  • Group
  • Message

Set ups

Several Webhooks where you can send or get data.

  • Group create
  • Post Message

A handful of Events and Translators to decide when and how perform transformations.

Events:

  • Message create_at
  • Group create_at

Translators:

  • Export Message
  • Update Model

Out of the Box Flows so you can focus on what's really important.

  • Message Create
  • Group Create

First steps

There's really only one thing you need to do: Map your models

  • Go to the Cenit IO Integrations menu.
  • Create a new Data type
    • Name it as you will.
    • Select the model you wish to send over to Slack (say 'im_chat.message').
    • Select the 'Slack' library
    • and the specific schema you want to use (say 'message.json').
    • On the Mappings tab set how you want your odoo's messages to translate to Slack's messages
    • Save your work.

That's it! Everytime you write a message in Odoo, a message will be create on your Slack account.

For more on mapping check the Documentation tab of the Cenit IO Integrations Client.
